1、类中使用构造函数，主要用于初始化；其中super()的作用是继承父类的引用;super（props）与super()的区别在于你是否需要在构造函数种使用this.props
constructor(props) {
        super(props)
        this.state = {
            username: ''
        }
}
2、'react-redux' 中的connect方法的作用:组件与store建立连接,组件内的props会添加store的dispatch方法;
当然也可以通过mapStateToProps和mapDispathToProps,将state和action添加到组件的props中，通过this.props访问

export default connect()(Login)
